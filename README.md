# 🍽️ Khan Cafe - Restaurant Web App

Khan Cafe is a modern restaurant web application built using React and Vite. It allows users to explore food items, view details, and manage a shopping cart with a smooth and responsive UI.

---

## 🔗 Repository

👉 [View Project on GitHub](https://github.com/AdilSami11/Khan-Cafe)

---

## 🚀 Features

* 🏠 Home page with hero section and “Top Picks”
* 🍕 Dynamic product listing using fake API
* 🔍 Product detail page with routing
* 🛒 Add to cart functionality
* ❌ Remove items from cart
* ⚡ Fast performance with Vite
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* CSS
